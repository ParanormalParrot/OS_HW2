 ### Вариант 32
 Разработать программу, которая на основе анализа двух входных ASCII-строк формирует на выходе две другие строки. В первой из выводимых строк содержатся символы, которых нет во второй исходной строке. Во второй выводимой строке содержатся символы, отсутствующие в первой входной строке (разности символов). Каждый символ в соответствующей выходной строке должен встречаться только один раз.

## Формат ввода и вывода
 Программа принимает целые числа m, n, k в качестве   коммандной строки.
В разделяемой памяти хранится массив целых чисел - номеров книг. При запуске программы он заполняется случайеыми числами. Далее создаются процессы - студенты - по одному на каждый ряд. Каждый процесс создаёт подмассив элементов массива из разделяемой памяти, в который входят все элементы из соостветствующего ему ряда. Далее подмассив сортируется при помощи сортировки вставками, причём поток выводит сообщение каждый раз, он находит место для книги. Когда массив отсортирован, поток передаёт отсортированные значения обратно в массив  в разделённой памяти и выводит соответствующее сообщение.
Когда все потоки отработали, основной поток уже целиком сортирует получившийс
